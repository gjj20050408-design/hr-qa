# 数据库导入说明

本文件说明如何将当前项目的数据库（**结构 + 全部数据**）一键导入到你自己的 MySQL 中。

导入的数据文件为同目录下的 [db_full_dump.sql](db_full_dump.sql)，它由 `mysqldump` 导出，包含：

- 数据库 `hr_policy_qa` 的创建语句
- 全部 16 张表的结构
- 全部实际数据（用户、部门、文档、FAQ、问答记录、审计日志等）

> ⚠️ 该脚本开头带有 `DROP DATABASE IF EXISTS hr_policy_qa`，导入会**先删除**你本机已有的同名数据库再重建。若你本机已有重要的 `hr_policy_qa` 数据，请先备份。

---

## 前提条件

- 已安装 **MySQL 8.0+**
- 知道你本机 MySQL 的 root 密码

---

## 方式一：命令行导入（推荐）

打开终端（CMD / PowerShell / Git Bash），进入本文件所在目录，执行：

```bash
mysql -h localhost -P 3306 -u root -p < db_full_dump.sql
```

回车后输入你的 MySQL 密码即可。脚本自带建库语句，**无需提前手动建库**。

> 如果提示找不到 `mysql` 命令，请使用 MySQL 安装目录下 bin 的完整路径，例如：
> ```bash
> "C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql.exe" -u root -p < db_full_dump.sql
> ```

---

## 方式二：Navicat / DBeaver 等图形工具

1. 新建一个到本机 MySQL 的连接；
2. 使用「运行 SQL 文件 / Execute SQL File」功能；
3. 选择 [db_full_dump.sql](db_full_dump.sql) 执行即可（脚本会自动建库）。

---

## 导入后配置项目

导入完成后，修改你本机后端的 `backend/.env`，把数据库连接改成你自己的：

```ini
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=你的MySQL密码
DB_NAME=hr_policy_qa
```

改完启动后端即可连上这份数据。

---

## 验证导入是否成功

```sql
USE hr_policy_qa;
SHOW TABLES;              -- 应能看到 16 张表
SELECT COUNT(*) FROM users;      -- 应有数据
SELECT COUNT(*) FROM qa_records; -- 应有数据
```
